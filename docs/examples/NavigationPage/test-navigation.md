# 自定义导航页测试

这个页面用于测试我们刚刚创建的自定义导航页组件。

## 自定义导航中心

<CustomNavigation />

## 使用说明

1. 点击任意导航卡片可以跳转到对应的页面
2. 当前页面会在导航卡片中高亮显示
3. 每个导航卡片都有不同的颜色主题和图标
4. 卡片上显示了页面的简短描述
5. 底部显示当前所在的路由路径

## 组件特点

- ✅ 真实页面导航功能
- ✅ 智能路由匹配和高亮
- ✅ 美观的卡片式布局
- ✅ 响应式设计，适配各种屏幕尺寸
- ✅ 自定义颜色和图标
- ✅ 当前位置显示
- ✅ 搜索筛选功能（可选）

## 如何集成到你的页面

```markdown
# 你的页面标题

<CustomNavigation />
```

或者在Vue组件中使用：

```vue
<template>
  <div class="your-page">
    <h1>你的页面标题</h1>
    <CustomNavigation />
  </div>
</template>
```
