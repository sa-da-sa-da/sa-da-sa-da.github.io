# 会考信息基础知识

## 一、基本键、组合键用法(很实用)

组合键：Ctrl、Shift、Alt，不能单独使用，只能和其它键组合使用。

- **Shift**: 
  - 上档字符键（录入键上上面的字符）
  - 选择多个连续的文件或文件夹
- **Ctrl**:
  - 选择多个不连续的文件或文件夹
  - 复制选中的对象
  - Ctrl+A: 全选
  - Ctrl+C: 复制
  - Ctrl+V: 粘贴
  - Ctrl+Home: 将光标移到文件首
  - Ctrl+End: 将光标移到文件尾
  - Ctrl+N: 新建
  - Ctrl+O: 打开
  - Ctrl+S: 保存
  - Ctrl+P: 打印
  - Ctrl+空格键：中/英文输入法转换
  - Ctrl+Shift：输入法之间的轮流切换
- **Alt**:
  - Alt+F4: 关闭打开的应用程序，多次使用可以一直到关机
  - Alt+回车: Excel单元格内换行
- **其他常用键**:
  - Caps Lock: 大写字母转换键
  - Num Lock: 键盘右侧数字区域锁定键
  - Tab键: 退格键，光标一次前进4位
  - Esc(escape)：取消、退出
  - Ctrl+Alt+Del：热启动（偶尔死机可以用）
  - Insert: 插入/改写状态，"改写"状态下打一字删后面一个字
  - Home：将光标移到当前行首
  - End：将光标移到当前行尾
  - PgUp：向上翻页
  - PgDn：向下翻页

## 二、编码（必修1P18-P24）

> 进制转换只是数值表示形式不同，值是相等的

### 1. 进制转换

- 二进制1位可表示两种状态(0或1)，即取值范围为0~1（必修一P24拓展）
- 十进制数X转换为N进制方法：将X除以N每次求余数，直到商为0为止，将余数倒序依次排列所得数
- N进制转换为十进制方法：每位上数字×N^m（m为该数字后面的位数，数字从最高位开始）,直到最后一位，将结果依次相加，所得数即为十进制数
- 示例：(11011)₂=1×2⁴+1×2³+0×2²+1×2¹+1×2⁰=27
- 位（bit）是信息的最小单位，一个字节（B）由8位二进制位组成，一般用来表示存储容量
- 一个汉字占两个字节

### 2. ASCII码(必修一P21)

- 每个字符在计算机中对应一个二进制数值，叫ASCII码值，字符叫ASCII码(美国标准信息交换代码)
- 由8位二进制构成
- "A"对应的ASCII值为65对应的二进制，后面的依次类推
- "a"对应的值为97对应的二进制，后面的依次类推
- 小写比相应的大写大32

### 3. 声音和图像数字化步骤和公式（熟记必修1P21-24）

- 数字化步骤：采样、量化、编码
- 未经压缩的音频文件数据量计算方法：
  ```
  数据量（单位:字节）=数据率×持续时间=（采样频率×量化位数×声道数）÷8×持续时间
  ```
- 未经压缩的位图图像的数据量计算方法：
  ```
  数据量（单位:字节）=图像分辨率×颜色深度÷8
  ```
- 以上单位为"B"，若须转换为KB、MB、GB，要除以依次1024

## 三、计算机系统（必修2 P45-P49）

- 1946年世界上第一台计算机（ENIAC）诞生（由美国的宾夕法尼亚大学研制成功）
- 世界上出现最早的网络为美国防部建的ARPANET网

### 1. 计算机发展阶段

- ①电子管计算机时代
- ②晶体管计算机时代
- ③中、小规模集成电路计算机时代
- ④大规模、超大规模集成电路计算机时代
- ⑤人工智能机

### 2. 计算机系统架构

- 计算机系统采用冯·诺依曼（美国数学家）体系，将计算机系统分为硬件系统和软件系统
- 计算机硬件五大部分：运算器、控制器、存储器、输入设备、输出设备

### 3. 中央处理器（CPU）

- 由运算器和控制器构成
- CPU的主频：是衡量计算机运算速度的重要指标，是计算机的重要部件，主要决定计算机的性能
- 单位是MHz、GHz

### 4. 内存

- **RAM（随机存取存储器）**:
  - CPU可直接访问
  - 断电内容就消失，所以我们要不断地进行保存操作
  - 保存到外存，外存断电后内容不会消失
- **ROM（只读存储器）**:
  - 如软件出厂前已固化在主板上的BIOS
  - 断电里面内容不会消失
  - 容量很小

### 5. 软件系统（必修2P58）

- **系统软件**:
  - 操作系统
  - 设备驱动程序
  - 语言处理程序等
  - 常见系统软件：Windows系统，unix系统，C语言，Python语言，汇编语言，Pasical语言等
- **应用软件**:
  - 为了某种特定的用途而被开发的软件，是一个特定的程序
  - 常用应用软件：WPS, office, Photoshop, 银行系统, flash软件, 看图软件, 各种办公软件, 定票系统等

## 四、网络基础（了解）

### 网络规划与结构

- 网络规划时可以将数字家庭系统网络涉及的设备按接入层、网络层、终端层进行划分(必修二P66)
- 网络的拓扑结构：总线型、星形、网络型
- 计算机网络拓扑结构根据其连线和节点的连接方式可分为以下几种类型：总线型、环形、星型、树形、网型

### 网络分类(必修二P70)

- 按传输介质分：有线和无线网络
- 按规模和地理范围分：
  - 个域网（PAN）
  - 局域网(LAN)
  - 城域网(MAN)
  - 广域网(WAN)

### 网络传输介质

常见的网络传输介质有：双绞线、同轴电缆、光纤等

### 网络协议

- 因特网的信息传递是按照统一的语言进行的，这个语言就是人们通常所称的因特网TCP/IP协议
- HTTP协议：Hyper Text Transfer Protocol（超文本传输协议）的缩写，是用于从万维网（WWW:World Wide Web）服务器传输超文本到本地浏览器的传送协议，是万维网的数据通信的基础
- HTTPS协议：即超文本传输安全协议，是一种通过计算机网络进行安全通信的传输协议

### IP地址(必修二P81-82)

- IP协议是为计算机网络相互连接进行通信而设计的协议
- 目前使用的有IPv4（32位）和IPv6（128位）两个版本，现有的是IPv4
- IP地址是一个32位的二进制数，通常被三个小数点分割为4个（0—255）"8位二进制数"(也就是4个字节)
- IPv4地址分为A类、B类、C类、D类、E类5种地址格式
  - A类适用于大型计算机网络
  - B类适用于中型计算机网络
  - C类适用于小型网络，一般用户都是C类
- 子网掩码不能单独存在，它必须和结合IP地址一起使用
- 网关作用好比古代国家的边关，对于小型局域网来说，通常路由器就是网关，路由器地址是网关地址

### 域名

- 域名（英语：Domain Name），由于IP地址具有不方便记忆并且不能显示地址组织的名称和性质等缺点，人们设计出了域名
- 通过网域名称系统（DNS，Domain Name System）来将域名和IP地址相互映射
- 常用顶级域名：.com .edu .gov .net等
- 两个字的后缀表示该域所在的国家或地区，例如：cn（中国大陆）、de（德国）、eu（欧盟）、jp（日本）、hk（中国香港）、tw（中国台湾）、uk（英国）、us（美国）
- 示例：www.zzedu.net.cn (郑州教育信息网)

### 网络带宽(必修二P79)

- 指单位时间内从网络中的某一点到另一点可传输的最大数据量
- 带宽越大，单位时间内的数据流量越大

### 物联网(必修二P91)

- 指物物相连中的互联网络
- 常见应用：智能交通、智能小区、数字博物馆、数字家庭、现代物流管理、食品安全监控、数字医疗等
- 物联网可以分为四层架构（理解并熟记）：感知层—传输层—数据处理层—应用服务层

### 1. 输入函数 input()

```python
input(<提示语>)
```
- 输入的内容均为str格式
- 提示语不执行，只是起到提示作用
- 输入完毕按Enter（回车键）
- 如果要参加加减乘除运算，必须使用int(input())或float(input())进行强制转换成整型或者浮点型

### 2. 输出函数 print()

```python
print(<提示语，>变量或常量列表<，end=''>)
```

- 各项之间用逗号隔开
