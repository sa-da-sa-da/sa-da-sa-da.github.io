## 【28】鸡兔同笼问题

《孙子算经》记载：“今雉（鸡）兔同笼，上有三十五头，下有九十四足，问雉兔各几何？”  
现在请你编写一个 Python 程序，要求用户输入鸡和兔的总头数和总脚数，计算出鸡和兔分别有多少？  
算法可以用流程图表示，完善如下流程图，将①②③处补充完整。

<MultipleChoiceQuestion 
  title="流程图中①应该补充为（   ）。"
  :options="[
    'i*4',
    'tou+1',
    'tou-i',
    'i+1'
  ]"
  :correctOptions="[3]"
  explanation="循环变量 i 需每次 +1，故①填 i+1。"
/>

<MultipleChoiceQuestion 
  title="流程图中②应该补充为（   ）。"
  :options="[
    'i*4',
    'tou-i',
    'tou+1',
    'i+1'
  ]"
  :correctOptions="[0]"
  explanation="计算兔的脚数：兔数量 = tou-i，兔脚数 = (tou-i)*4，即 i*4 表示鸡脚数，②处需计算总脚数表达式，故填 i*4。"
/>

<MultipleChoiceQuestion 
  title="流程图中③应该补充为（   ）。"
  :options="[
    'tou-i',
    'i+1',
    'i*4',
    'tou+1'
  ]"
  :correctOptions="[0]"
  explanation="若脚数匹配，则兔的数量 = 总头数 - 鸡的数量 i，故③填 tou-i。"
/>
