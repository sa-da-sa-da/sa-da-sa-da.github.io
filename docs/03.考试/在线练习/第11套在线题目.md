示例：输入数据：671 1167122

```python
def ok(x,y):
    if x in y:
        return True
    else:
        return False
mi=input()
s=input()
hw=""
for i in range(1,len(mi)+1):
    hw+=mi[-i]
if ok(mi,s) or ok(hw,s):
    print("Yes")
else:
    print("No")

```
