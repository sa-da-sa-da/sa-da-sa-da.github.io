<a href="https://ossaiimages.sakaay.com/ai-image/sakaay/%E8%AF%95%E9%A2%98%E5%9B%BE%E7%89%87/excel/9-excel.xlsx" target="_blank">点击下载文件，打开“销售业绩表.xlsx”文件。</a>

1. 在“销售业绩表”工作表中，将A1:G1单元格区域合并居中。  
2. 使用函数计算每位员工的“总销售额”，将结果填入F3:F19单元格中。  
3. 使用函数计算每个季度的“最高销售额”，将结果填入B20:E20单元格中。  
4. 使用“条件格式”之“突出显示单元格规则”，将“总销售额”介于3100和3200的数据格式设置为“浅红填充色深红色文本”。  
5. 使用RANK函数计算“总销售额”按降序排列的大小排名，填入G3:G19单元格中。  
   【提示：排名函数RANK()的作用是返回某数字在一列数字中相对于其他数值的大小排名。例如：=RANK(D3,$D$3:$D$10)表示D3单元格数据在被锁定的单元格区域D3:D10的数据中相对于其他数据按降序排列的大小排名】  
6. 保存文件并关闭WPS。
