# 填空题组件示例

本文档展示了新创建的 `FillInTheBlank` 组件的使用方法和功能。这个组件支持单空、多空填空题，并且具有答题验证、结果显示和解析功能。

## 基本使用方法

### 单空填空题

```md
<FillInTheBlank
  title="计算机网络中，IP地址的长度是{blank}位。"
  :correctAnswers=["32"]
  explanation="IPv4地址由32位二进制数组成，通常用点分十进制表示。"
/>
```

<FillInTheBlank
  title="计算机网络中，IP地址的长度是{blank}位。"
  :correctAnswers=["32"]
  explanation="IPv4地址由32位二进制数组成，通常用点分十进制表示。"
/>

### 多空填空题

```md
<FillInTheBlank
  title="在计算机系统中，{blank}是运算器和控制器的合称，而{blank}是计算机的核心部件。"
  :correctAnswers=["CPU", "中央处理器"]
  explanation="CPU（中央处理器）由运算器和控制器组成，是计算机的核心部件，负责执行指令和处理数据。"
/>
```

<FillInTheBlank
  title="在计算机系统中，{blank}是运算器和控制器的合称，而{blank}是计算机的核心部件。"
  :correctAnswers=["CPU", "中央处理器"]
  explanation="CPU（中央处理器）由运算器和控制器组成，是计算机的核心部件，负责执行指令和处理数据。"
/>

## 组件属性说明

`FillInTheBlank` 组件支持以下属性：

| 属性名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| title | string | 是 | - | 题目文本，可以包含 `{blank}` 占位符来标记空格位置 |
| correctAnswers | string[] | 是 | - | 正确答案数组，每个元素对应一个空格的答案 |
| explanation | string | 是 | - | 题目的解析说明 |
| images | string[] | 否 | [] | 题目图片URL数组 |
| index | number/string | 否 | - | 自定义题目序号 |
| showIndex | boolean/'withDot'/'withColon' | 否 | true | 序号显示方式 |
| placeholder | string | 否 | "请输入第X空的答案" | 输入框占位符文本 |
| caseSensitive | boolean | 否 | false | 答案是否大小写敏感 |

## 高级使用示例

### 带图片的填空题

```md
<FillInTheBlank
  title="图中显示的设备是{blank}，它的主要功能是{blank}。"
  :correctAnswers=["路由器", "连接不同网络"]
  explanation="路由器是连接不同网络的网络设备，能够根据数据包的目的地址进行路由选择和转发。"
  :images=["https://example.com/router.jpg"]
/>
```

### 自定义序号和显示格式

```md
<FillInTheBlank
  title="HTML的全称是{blank}。"
  :correctAnswers=["HyperText Markup Language"]
  explanation="HTML是超文本标记语言，用于创建网页的标准标记语言。"
  index="1-1"
  showIndex="withDot"
/>
```

<FillInTheBlank
  title="HTML的全称是{blank}。"
  :correctAnswers=["HyperText Markup Language"]
  explanation="HTML是超文本标记语言，用于创建网页的标准标记语言。"
  index="1-1"
  showIndex="withDot"
/>

### 大小写敏感的填空题

```md
<FillInTheBlank
  title="JavaScript中，用于声明变量的关键字有var、let和{blank}。"
  :correctAnswers=["const"]
  explanation="JavaScript中，var、let和const都是用于声明变量的关键字，其中const用于声明常量。"
  :caseSensitive=true
/>
```

## 使用注意事项

1. 题目文本中的空格位置需要使用 `{blank}` 占位符进行标记，或者不使用占位符（组件会在题目末尾自动添加空格提示）。
2. `correctAnswers` 数组的长度必须与题目中的空格数量一致。
3. 默认情况下，答案验证是不区分大小写的，可以通过设置 `caseSensitive=true` 来启用大小写敏感验证。
4. 组件会自动跟踪题目序号，也可以通过 `index` 属性自定义序号。
5. 提交答案后，可以查看答题结果和解析，点击"重新答题"按钮可以重置题目。
