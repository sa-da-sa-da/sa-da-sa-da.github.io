# 隐私政策

一、信息收集

当您访问本站时，我们可能会自动收集如下信息：

您的IP地址、浏览器类型、操作系统及访问时间等基本信息

通过第三方服务（如 Google AdSense）可能收集到的匿名统计信息

本站不会主动要求您提供任何个人身份信息（如姓名、身份证号等）。


二、Cookie 与第三方广告

本站集成了 Google AdSense 广告服务。Google 及其合作方可能会通过Cookie、网络信标等方式收集、使用您对广告的访问记录和行为以确定广告内容的相关性和效果。

Google等第三方广告商可能会根据您的兴趣投放广告。

您可以访问 Google广告政策 以了解详细信息，也可以在Google广告设置中管理个性化广告。

您可以在浏览器设置中禁用Cookie，但这可能影响网站部分功能体验。


三、信息用途

用于统计网站流量及优化服务用户体验

用于广告投放与广告效果分析（如Google AdSense）

四、信息保护

我们承诺对用户数据采取合理安全措施，防止数据丢失、滥用和篡改。本站不会主动将您的信息用于其他商业目的或泄露给第三方，除法律法规要求或执法机关要求。

五、隐私政策更新

我们可能会不定期更新本政策。请您定期访问本页面，及时了解最新隐私政策内容。

如您对隐私政策有任何疑问或建议，请联系我们。
