## Visual Studio Code 安装

我们推荐使用[Visual Studio Code](https://code.visualstudio.com/)作为代码编辑器。点击链接并点击下载按钮，如下图所示：
![vscode](https://file.sakaay.com/sakaay/vex%20pic/download-visual-studio-code.webp)

## PROS 安装

[PROS](https://pros.cs.purdue.edu/)提供了构建和上传程序所需的工具。它可以作为 VSCode 的[扩展](https://marketplace.visualstudio.com/items?itemName=sigbots.pros)进行安装。只需在 VSCode 左侧边栏底部点击方块，搜索“PROS”，然后点击“安装”，如下图所示：
![pros](https://file.sakaay.com/sakaay/vex%20pic/install-pros.webp =400*400)

扩展下载完成后，你应该会看到如下图所示的通知。点击安装并等待安装完成：
![pros-toolchain](https://file.sakaay.com/sakaay/vex%20pic/download-pros-toolchain.webp)

工具链安装完成后，你可以创建一个新的 PROS 项目。在边栏中选择 PROS 标志，点击“创建项目”，为项目选择一个空文件夹，给它一个合适的名称，例如“1010N-Spin-Up”，选择“v5”，对于内核版本选择“3.8.3”。右下角会弹出一个通知，通知你项目正在创建中。VSCode 准备好后会打开一个新窗口。会有一个弹出窗口询问是否“信任”该文件夹，点击“是”。现在你已经安装了 PROS 并创建了你的第一个项目！

## clangd 安装

[clangd](https://marketplace.visualstudio.com/items?itemName=llvm-vs-code-extensions.vscode-clangd)是一个代码检查工具，提供代码补全功能，对任何程序员来说都是必不可少的。我们推荐这个而不是[C/C++](https://marketplace.visualstudio.com/items?itemName=ms-vscode.cpptools)扩展，因为它速度更快。

要安装 clangd，重复安装 PROS 扩展的步骤，但搜索“clangd”而不是“PROS”。安装完成后，打开项目中的“main.cpp”文件。clangd 会在右下角出现一个通知，说明它需要安装。点击“安装”并等待安装完成。如果通知没有出现，请确保你已经安装了 clangd 并重新打开“main.cpp”。现在你已经安装了 clangd！

## LemLib 安装

要安装 LemLib，首先需要打开 PROS 集成终端。这可以通过点击屏幕左侧的 PROS 图标并点击“集成终端”来完成。

然后，将以下命令复制/粘贴到你的命令行中。这将告诉 PROS 在哪里可以找到 LemLib，并将 LemLib 的最新稳定版本应用到你的项目中：
```
pros c add-depot LemLib https://raw.githubusercontent.com/LemLib/LemLib/depot/stable.json # 添加 LemLib 的稳定仓库
pros c apply LemLib # 应用 LemLib 的最新稳定版本
```

最后，将以下内容放在你的“main.cpp”文件的顶部：
```c++
#include "lemlib/api.hpp"
```

现在你已经安装了 LemLib，并且准备好配置你的机器人了！
