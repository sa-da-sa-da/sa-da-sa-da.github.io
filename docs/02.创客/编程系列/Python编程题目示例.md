# Python编程题目示例

本页面展示如何使用ID模式加载不同的Python编程题目示例。通过指定`problemId`属性，我们可以轻松在文档中嵌入特定的编程题目代码。

## 功能说明

使用ID模式可以带来以下好处：

- **代码与文档分离**：Python代码存储在中央代码库中，便于维护和更新
- **一致的用户体验**：在多个文档中重复使用相同的代码示例
- **简化Markdown编写**：只需指定题目ID，无需在Markdown中直接编写Python代码
- **自动加载预设输入**：题目可以包含默认输入参数，简化用户操作

## 在您的文档中使用

在您自己的Markdown文档中使用ID模式非常简单，只需添加以下代码：

```markdown
<ClientOnly>
  <PythonEditor problemId="题目ID"/>
</ClientOnly>
```

将`题目ID`替换为上面列表中任一有效的题目ID，即可在您的文档中嵌入相应的Python题目代码。

## 注意事项

- 首次加载Python环境可能需要一些时间
- 您可以随时修改加载的代码，但不会影响原始代码库中的内容
- 如果需要自定义题目，可以联系管理员将新题目添加到代码库中
